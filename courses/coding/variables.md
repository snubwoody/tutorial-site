title: Variables
---

A variable is a like a container for data.

```python
name = "Hello world"
```

## Data types
There's 10 data types in python:
- `int`
- `string`
- `boolean`

## Changing variables
Variables can be changed at any time.


```python
count = 0
print(count)
count = 1
print(count)
```
